# Plantilla · Especificaciones visuales para piezas derivadas

> Specs técnicas reutilizables para diseñador o VA al producir Shorts, carruseles, citas y thumbnails.
> Origen: doc #3 `youtube-podcast-estrategia-canal.md` §8.4 + §9.2 + doc #5 `reciclaje-contenido-pipeline.md` §4.

---

## Sistema de identidad visual TWIM (común a todas las piezas)

| Atributo | Valor |
|---|---|
| Color verde oscuro | `#173D30` |
| Color verde medio | `#265C4B` |
| Color beige | `#C2A78B` |
| Color fondo claro | `#FDFCFA` |
| Tipografía principal | Barlow Condensed (Google Fonts) |
| Pesos usados | Regular, SemiBold, Bold, ExtraBold |
| Logo principal | `LOGO.png` (en raíz del repo) |
| Logo alternativo | `logo-mindworld.png` |
| Cover podcast | `podcast-cover.png` (en raíz del repo) |
| Firma fija en todas las piezas | `Daniel Orozco Abia · CV11515 · Valencia` |

> Nunca usar emojis en piezas visuales. Nunca usar caras gritando, flechas rojas, círculos amarillos ni efectos zoom dramáticos.

## 1 · Thumbnail YouTube · Episodio long-form

| Atributo | Valor |
|---|---|
| Dimensiones | 1280 x 720 px (16:9) |
| Peso máximo | 2 MB |
| Composición | Cover unificado del podcast + título corto sobreimpreso |
| Color de fondo | Verde oscuro `#173D30` |
| Color del título | Beige `#C2A78B` |
| Tipografía título | Barlow Condensed Bold, ~80-100 px |
| Longitud título visible | 3-5 palabras (no el título completo del vídeo) |
| Posición título | Tercio superior o central, alineado a izquierda |
| Logo TWIM | Esquina inferior derecha, ~80 px de altura |
| Test legibilidad | Reducir al 30 % y comprobar que título se sigue leyendo |

**Nada en el thumbnail:** foto de Daniel reaccionando, emoji, flecha, círculo, signo de exclamación grande.

## 2 · Shorts y Reels (Pieza #3 · 3-5 por episodio)

| Atributo | Valor |
|---|---|
| Aspect ratio | 9:16 |
| Resolución | 1080 x 1920 px |
| Duración | 30-60 s · óptimo 45 s |
| Frame inicial (cover frame) | 1-2 s con título 3-5 palabras antes de que arranque audio |
| Color de fondo cover frame | Verde oscuro `#173D30` |
| Tipografía título cover | Barlow Condensed Bold, ~120-140 px |
| Color título cover | Beige `#C2A78B` |
| Subtítulos | Quemados en vídeo (no CC opcional) |
| Color subtítulos | Beige `#C2A78B` con caja semitransparente verde oscuro |
| Tamaño subtítulos | 60-72 px |
| Posición subtítulos | Centro vertical, dejando espacio inferior para barra de UI de IG/YouTube |
| Outro | 1-2 s con logo TWIM + handle `@daniorozcopsicologo` |
| Música | Sin música, o muy suave (-30 dB respecto a la voz) |
| Hashtags caption | 3-5 (marca + pilar) |

**Plataformas destino:** YouTube Shorts + IG Reels. Misma pieza, dos canales.

## 3 · Carrusel IG (Pieza #4 · sistema A1 del PR #94)

| Atributo | Valor |
|---|---|
| Dimensiones por slide | 1080 x 1350 px (4:5 vertical) |
| Número de slides | 5-7 |
| Slide 1 (cover) | Título del concepto en Barlow Condensed XL (~140 px), fondo verde oscuro `#173D30`, texto en beige `#C2A78B` |
| Slides 2-6 (cuerpo) | Layout simétrico beige sobre verde medio `#265C4B`, una idea por slide |
| Frase por slide intermedio | 8-12 palabras |
| Tamaño frase intermedios | ~80-100 px Barlow Condensed SemiBold |
| Slide final (CTA) | Pregunta abierta + CTA suave a perfil de Daniel o newsletter |
| Pie consistente | "Daniel Orozco Abia · CV11515" en cada slide, esquina inferior, ~28 px |
| Logo TWIM | Solo en slide 1 y slide final, esquina superior derecha |

**Sistema A1 = carrusel temático.** Es el formato del PR #94 §11.3 que aquí se reutiliza derivado del episodio.

## 4 · Cita visual IG (Pieza #5 · sistema A2 del PR #94)

| Atributo | Valor |
|---|---|
| Dimensiones | 1080 x 1080 px (1:1 cuadrado) |
| Composición | Frase memorable centrada |
| Longitud frase | Máximo 18 palabras |
| Tipografía | Barlow Condensed XL (~120-160 px), peso Bold o ExtraBold |
| Color fondo | Verde oscuro `#173D30` o variante invertida fondo claro `#FDFCFA` |
| Color texto | Beige `#C2A78B` sobre verde · Verde medio `#265C4B` sobre fondo claro |
| Atribución | "TWIM Podcast E{n} · {fecha}" en esquina inferior, ~24 px |
| Logo TWIM | Esquina superior izquierda o central, ~60 px |

**Sistema A2 = cita visual.** Una frase del episodio, sin cuerpo de explicación.

---

## 5 · Stories IG (uso opcional, no es Pieza fija)

| Atributo | Valor |
|---|---|
| Dimensiones | 1080 x 1920 px (9:16) |
| Duración por story | ≤ 15 s si es vídeo, ≤ 7 s lectura si es estática |
| Uso típico | Repostear cita visual, anunciar episodio nuevo, mostrar back-stage de producción |
| Tono | Más directo y espontáneo que feed |
| Sticker recomendado | Pregunta abierta o encuesta · 1 por story |

## Checklist genérico pre-publicación de cualquier pieza visual

- [ ] Paleta correcta (verde oscuro / verde medio / beige / fondo claro).
- [ ] Tipografía Barlow Condensed (no se ha colado Arial / Helvetica).
- [ ] Sin emojis.
- [ ] Sin código visual de Shorts genéricos (zoom, flechas, palabras parpadeando).
- [ ] Firma "Daniel Orozco Abia · CV11515" presente cuando aplique.
- [ ] Test de legibilidad: pieza reducida al 30 % se sigue leyendo.
- [ ] CTA coherente con el pilar y con la fase del embudo TWIM.
- [ ] Verificación final: cabe esto en la voz del libro Los engranajes de la mente. Si no, fuera.
